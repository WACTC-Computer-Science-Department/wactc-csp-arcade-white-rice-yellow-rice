# CSP Arcade Project
# White rice, Yellow Rice Team
This repository serves as a template for teams to create their own classic arcade games as part of the Computer Science Principles (CSP) course using p5.js.

## Project Overview




Students will work in teams to code a classic arcade game using p5.js and collaborative coding practices on GitHub Education.

## Getting Started

1. **Create a Team Repository**: Each team should create their own repository from this template.
   - Go to GitHub and create a new repository
   - Use this repository as a template
   - Name it appropriately (e.g., `Teacher Test Team`)

2. **Technology Stack**:
   - **p5.js**: JavaScript library for creative coding and graphics
   - Runs in web browsers
   - Includes built-in functions for drawing, animation, and interaction

3. **Game Requirements**:
   - Implement a classic arcade game (Pong, Space Invaders, Pac-Man, etc.)
   - Include game mechanics, scoring, levels
   - Use version control for collaboration
   - Document your code

## Repository Structure

```
team-repo/
├── index.html         # Main HTML file with p5.js
├── sketch.js          # Main game code
├── README.md          # Project documentation
├── assets/            # Images, sounds, etc.
└── docs/              # Additional documentation
```

## Collaboration Guidelines

- Use GitHub Issues for task tracking
- Create branches for new features
- Use pull requests for code reviews
- Commit frequently with descriptive messages
